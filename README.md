# AI Resume Analyzer

A beginner-friendly Python project that analyzes resumes and generates a simple report.

## Features

* Reads resume data from a text file
* Detects skills
* Identifies missing skills
* Calculates a resume score
* Generates a resume rating
* Provides improvement suggestions
* Saves analysis results to a report file

## Technologies Used

* Python
* File Handling
* Lists
* Loops
* Conditional Statements

## Example Output

Skills Found:

* Python
* SQL
* Git

Missing Skills:

* Java
* C++

Resume Score: 70/100

Resume Rating: Good

Suggestions:

* Try learning: Java, C++

## How to Run

```bash
python analyzer.py
```
